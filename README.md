# Cost_Statement_Change

설계변경 내역서 자동화 프로그램

설계변경 내역서 `.xlsx` 파일에서 금액 수식이 있는 데이터 행을 `당초/변경` 쌍으로 자동 변환하는 Python GUI 프로그램입니다.

## 실행

```cmd
D:\Coding\Cost_Statement_Change\run_app.bat
```

화면에서 엑셀 파일을 선택한 뒤 처리할 시트와 `당초/변경` 열을 삽입할 위치를 선택하고 `자동화 실행`을 누르면 됩니다.

## 처리 규칙

- 원본 파일은 수정하지 않고 `원본파일명_설계변경자동화.xlsx` 복사본을 만듭니다.
- 시트명은 `내역서`, `내역`을 우선 선택합니다.
- 헤더에서 `노무비`, `재료비`, `경비`, `금액` 열을 자동 탐색합니다.
- 금액 열에 수식이 있는 행 아래에 변경 행을 삽입합니다.
- 원래 행에는 `당초`, 새 행에는 `변경`을 입력합니다.
- 새 행은 원래 행의 값, 수식, 스타일을 복사하고 글자색을 빨간색으로 바꿉니다.
- 변경 행의 수식(금액 열 제외): 다른 시트 참조는 당초 행과 동일하게 유지하고, 같은 시트에서 당초 행을 가리키는 참조는 한 행 아래의 변경 행 참조로 바꿉니다 (당초 `A5` → 변경 `A6`). 헤더 등 고정 셀 참조는 그대로 둡니다.
- 새 행의 금액 수식은 `노무비+재료비+경비` 참조로 다시 설정합니다.
- 열/행 삽입으로 위치가 바뀐 시트 내 모든 수식(데이터 행 밖 포함)과 다른 시트의 내역서 참조 수식을 자동 보정합니다.
